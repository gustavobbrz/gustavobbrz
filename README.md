# ⚙️ Projeto em Destaque

## 🧠 loyctl — DevOps Operational CLI

CLI operacional desenvolvida para **monitorar e escalar fluxos de processamento** baseados em filas.

Ela permite que operadores DevOps:

* analisem backlog de filas
* detectem gargalos de processamento
* distribuam cargas entre servidores
* escalem containers automaticamente

### Exemplo de uso

```bash
./loyctl
```

```text
LOYCTL — DevOps Operational CLI

[ OPERACIONAL ]

1) Zerar flows
2) Inativar jobs
3) Delayed de jobs
4) Limbo

[ MONITORAMENTO ]

5) Status dos flows
6) Jobs FAILED
7) Escalar flows
8) Analisar filas

[ OTIMIZAÇÃO ]

9) Escalar filas grandes
```

### Radar operacional de filas

```text
FASE 1 — CAPTURA DE DADOS

Coletando Petição... [OK]
Coletando Consulta... [OK]
Coletando Movimento... [OK]

FASE 2 — PREVISÃO DE ESCALA

CLIENTE        SISTEMA        JOBS      CONTAINERS

BancoXPTO      pje            2100      12
BancoABC       eproc          1500      8
BancoDEF       pje            600       4

Distribuição planejada:

FLOW30 → 12 containers
FLOW75 → 8 containers
FLOW78 → 4 containers
```
